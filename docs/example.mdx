# 代码块示例
1. 不含标题的代码块
```go
func main(){
    fmt.Println("Hello world!")
}
```
2. 含标题的代码块
```go title="main.go"
func main(){
    fmt.Println("Hello world!")
}
```

3. 行高亮
```go {2} title="main.go"
func main(){
    fmt.Println("Hello world!")
}
```
