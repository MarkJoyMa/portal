## 概述

本教程使用 Protocol Buffers 语言的 proto3 版本，为 Go 程序员提供了使用 Protocol Buffers 的基本介绍。通过创建一个简单的示例应用程序，它向您展示了如何

- 在 .proto 文件中定义消息格式。
- 使用协议缓冲区编译器。
- 使用 Go protocol buffer API 来写入和读取消息。
